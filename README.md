# ✅ TaskFlow

TaskFlow is a modern, responsive task management web application built as part of the CS50 Homepage project. It allows users to manage their daily tasks efficiently with a clean UI and smooth interactions.

---

## 🚀 Features

* 📝 Add, delete, and manage tasks
* ✅ Mark tasks as completed by clicking anywhere
* 📌 New tasks appear at the top
* 💾 Data persists using browser localStorage
* 📊 Live task counter
* 🎨 Responsive design using Bootstrap
* 📄 Multi-page website (Home, Tasks, Features, Contact)

---

## 🧠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

---

---

## 💡 How It Works

* Tasks are stored in the browser using `localStorage`
* Default tasks are loaded on first visit
* Users can:

  * Add new tasks
  * Mark tasks as completed
  * Delete tasks
* The UI updates dynamically using JavaScript
